# A/B Test Plan: CTA Redesign for FILMLY

**Objective:** Test whether replacing the current popcorn-centered recommendation CTA with a more content-driven layout improves user interaction interest in the recommendation feature, hence sign up.  

## Variant A
**Current CTA section:**
- popcorn illustration in the center  
- headline: Stop scrolling. Start watching.  
- supporting subtitle  
- CTA button: Sign in to FILMLY  

## Variant B
**Redesigned CTA section:**
- left side: headline and subtitle  
- right side: several movie recommendation cards  
- same main CTA  

## Hypothesis
If we add the early preview of the recommendation feature near the CTA on the main page, the number of users who sign up will increase by 5% percentage points, because they will see the value of the feature more clearly and visually.  

## Target audience
We need two groups of non-registered users. The control group will see the original page - Variant A (no change). The test group will see the redesigned version - Variant B.  

Around 50% of traffic will be allocated to Variant A and 50% to Variant B.  

## Primary metric
- Click Rate.  

## Additional metrics
- Sign Up Rate. It is not the main metric as it does not properly reflect the goal since users can drop signing up for other reasons  
- Clicks on recommendation cards  
- Bounce rate  

## Traffic assumptions
The startup currently has a relatively small volume of new users.  
The main page is visited by around 100–120 non-registered users per day.  

## Sample size calculation
- Baseline conversion rate: 5%  
- Minimum Absolute Detectable Effect: 5%  
- Statistical Power: 80%  
- Significance Level: 5%  

According to the online calculator, a sample size of each group should be 336 participants. 672 rounded in total.  

## Test duration
Test duration (days) = Sample size / Average daily traffic  
Test duration (days) = 672 / 100 = 6.7 days  

The data  will be gathered for 7 days to cover the full week and ensure sufficient sample size.
